# Mathe-Miezi

If my niece is going to use an AI for her math homework anyway, it might as well be one that does not hand her the solution on a silver platter.

Mathe-Miezi is a cat-themed math tutor that is aimed to guide a high-schooler through math problems with hints and encouragement.

Built with Streamlit and GPT-4o-mini, available in Italian and German.
